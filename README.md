Simple Ionic Timer
==================

An ionic application.
A simple timer with an ending beeping sound.

## Using this project

**Android**
```bash
$ npm install -g ionic cordova
$ ionic cordova build android
$ ionic cordova run android
```

**IOS**
```bash
$ npm install -g ionic cordova
$ ionic cordova build ios
```

Then, deploy with XCode.
